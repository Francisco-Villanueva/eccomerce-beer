E-commerce: DEVBOOKS

![Logo](devbooks.png)

Descripción:

DevBooks es una aplicacion de libros que ofrece una amplia selección de libros de programación, desarrollo web y tecnología. Nuestra aplicacion te permite explorar y comprar libros relacionados con la tecnología de manera fácil y conveniente.

Funcionalidades MUST HAVE:

- Podes registrarte y acceder como administrador o usuario.
- Loguearte y Mantener sesión abierta ante un cierre del browser o refresh
- Buscar una amplia variedad de libros de programación y tecnología.
- Filtrar libros por categorias.
- Agregar editar o eliminar libros por título, autor o categoría si sos administrador
- Mira detalles de los libros, como la descripción, el autor y el precio.
- Agrega libros a tu carrito de compras.
- Realiza pedidos seguros y protegidos.
- Recibe notificaciones sobre el estado de tu pedido por correo electrónico.

Tecnologías Utilizadas:

- HTML, CSS y JavaScript para la interfaz de usuario.
- Material UI y Bulma para el diseño de la aplicacion.
- Node.js y Express.js para el backend.
- Sequelize para la base de datos.
- React para el frontend.
- ANTD para los mensajes de alertas.
- Google books API para obtener información de libros. [API de Libros](https://developers.google.com/books?hl=es-419)
- JWT para la autenticación de usuarios.

Instalación:

1- Clona este repositorio en tu máquina local:

- git@github.com:Francisco-Villanueva/eccomerce-book.git

2- Instala las dependencias del proyecto con:

- npm install tanto para la carpeta API y CLIENT e inicializar el backend con npm start y el frontend con npm run dev.
